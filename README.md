\# Telegram Currency Converter Bot



A simple Telegram bot that converts currencies using real-time exchange rates.  

Built with \*\*Python\*\*, \*\*pyTelegramBotAPI\*\*, and \*\*requests\*\*.



---



\## Features



\- Converts from one currency to another (e.g., USD → RUB)

\- Supports multiple currencies (доллар, евро, рубль, биткоин, эфир)

\- Parses real-time exchange rates from \[CryptoCompare API](https://min-api.cryptocompare.com/)

\- Handles user input errors gracefully

\- Clean architecture with modular files



---



\## Commands



\- `/start` or `/help` — show usage instructions  

\- `/values` — list of available currencies
